About Me

My name is Toumas, and this website was created as part of an academic assignment to demonstrate the use of GitHub Pages with the Jekyll static site generator.

The goal of this project is to gain hands-on experience in creating and deploying a static website using modern development tools and cloud-based hosting.

About This Project

This website is hosted publicly using GitHub Pages and is built using Jekyll, which allows static content to be generated from simple HTML files.

The project includes multiple pages, navigation between pages, and the use of a supported Jekyll theme configured through the _config.yml file.

Skills Developed

  • Using GitHub repositories and version control
  • Deploying websites with GitHub Pages
  • Configuring Jekyll themes
  • Creating multi-page static websites using HTML

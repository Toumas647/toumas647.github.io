Blog

This blog page is used to document learning reflections and experiences gained during the development of this GitHub Pages website.

My First Post

In this assignment, I learned how to create a GitHub Pages website using the Jekyll static site generator. I also learned how to configure a theme and deploy the site publicly.

Challenges Faced

One of the challenges encountered was understanding how Jekyll processes files and how themes are applied using the _config.yml file. This was resolved by following the official GitHub documentation.

What I Learned

  • How GitHub Pages automatically builds and deploys sites
  • The role of Jekyll front matter in HTML files
  • How static websites differ from dynamic web applications

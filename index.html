Welcome to My GitHub Pages Website

This website was developed as part of Assignment 1 to demonstrate the use of GitHub Pages with the Jekyll static site generator.

About This Website

The purpose of this website is to showcase a static multi-page website created using GitHub Pages. Jekyll is used as the static site generator, allowing content to be written in HTML while still benefiting from layouts and themes.

The site uses an officially supported GitHub Pages Jekyll theme, which provides consistent styling and layout across all pages.

Technologies Used

  • GitHub Pages for website hosting
  • Jekyll static site generator
  • HTML for page structure
  • YAML configuration using _config.yml

Site Structure

This website consists of multiple pages, each serving a specific purpose:

  • Home – Overview of the website
  • About – Information about the author and project
  • Projects – Details of work completed
  • Contact – Contact information
  • Blog – Reflections and learning notes

Learning Outcomes

Through this assignment, I learned how to:

  • Create and configure a GitHub Pages website
  • Use Jekyll themes and layouts
  • Deploy a public static website using GitHub
  • Organize content across multiple pages
